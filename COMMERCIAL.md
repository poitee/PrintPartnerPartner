# Commercial licensing

Print Partner’s source code is under the **[PolyForm Noncommercial License 1.0.0](LICENSE)**. That license covers everyday noncommercial use, modification, and sharing.

This page is only about uses that **PolyForm does not allow** without a separate written agreement.

---

## Usually fine without a commercial license

- Running Print Partner in your shop or studio to manage jobs, kits, and exports
- **Selling the physical parts** you print (the license applies to the **software**, not your products)
- Personal, educational, hobby, and typical nonprofit use as described in [LICENSE-SUMMARY.md](LICENSE-SUMMARY.md)

---

## Needs written permission

Contact the copyright holder before:

- Selling Print Partner or a fork (installer, app store, paid download, etc.)
- Bundling Print Partner inside another product you sell
- Offering Print Partner (or your derivative) as a **SaaS**, managed service, or paid API
- Any use that is not **noncommercial** under PolyForm’s definition in [LICENSE](LICENSE)

---

## How to request a commercial license

Open a **private** GitHub issue or discussion on the project repository:

https://github.com/poitee/PrintPartnerPartner

Describe how you plan to use or distribute the software. Ko-fi support is appreciated but **does not** replace a commercial license.
